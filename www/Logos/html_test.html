How to use the Brick:

 

Step 1: Configuration

Navigate to Configuration Tab to load the required prerequisites

 

Step 2: Scenario Planning

Proceed to Scenario Planning Tab to Generate and Run Simulation / Optimization Scenarios

 

Step 3: Scenario Comparison

For comparing the created scenarios, navigate to Scenario Comparison Tab

 

Step 4: Recommendations

For insights based on the created scenarios, navigate to Recommendations Tab
